# Дизайн: замена блока «Манифест» на блок-врезку «Один правильный разговор...»

## Что меняем

Блок `#manifesto` (тёмный, текст «Победа — это не победа над конкурентами...») полностью удаляется из проекта — и из разметки, и из стилей. На его месте (между `#includes` и `#format`) появляется новая секция `#statement` — источник: присланный пользователем референс-скриншот (светлый фон, чёрный текст, красные акценты, слово в овале от руки).

## Контент

Текст используется буквально как на референсе, без изменений и без CTA-кнопки (как и в текущем `.manifesto`, у которого кнопки тоже нет):

> Один **правильный** [плашка-заливка]
> **разговор** [акцентный текст] может стоить
> **больше** [обведено от руки] года попыток
> в одиночку.

## Разметка

```html
<section id="statement" class="statement">
  <div class="container statement__inner">
    <span class="statement__bar" aria-hidden="true"></span>
    <p class="statement__text">
      Один <span class="statement__highlight statement__highlight--fill">правильный</span><br>
      <span class="statement__highlight statement__highlight--text">разговор</span> может стоить<br>
      <span class="statement__highlight statement__highlight--circle">больше<svg class="statement__circle" viewBox="0 0 200 90" aria-hidden="true"><path d="M14,46 C10,24 34,9 76,7 C124,5 178,9 188,30 C198,52 172,73 110,78 C58,82 14,74 12,54" fill="none" stroke="#E31E24" stroke-width="6" stroke-linecap="round"/></svg></span> года попыток<br>
      в одиночку.
    </p>
  </div>
</section>
```

Секция — обычный полноширинный блок (как `.hero`, `.founder`, бывший `.manifesto`): фон задаётся на `.statement`, контент ограничен через `.container`, без full-bleed хаков.

## Визуальный стиль

Палитра и вес повторяют существующий блок `.growth` (та же светлая пауза в нарративе сайта, тот же приём):

- Фон секции: `#F7F7F8`, текст: `#0A0A0C`, акцент: `#E31E24` — цвета-литералы, без переопределения `--color-gold` (по аналогии с `.growth`, это разовая светлая секция, не часть тёмного красно-чёрного паттерна hero/founder).
- `font-family: var(--font-sans)` (Inter), `font-weight: 800`, без капса.
- Размер текста: `1.75rem` на мобильных → `2.5rem` от 768px → `3rem` от 1024px (немного меньше, чем `.growth__text`, так как тут на одну строку больше).
- Вертикальный паддинг секции — `var(--space-6)`, как у соседних секций.

Акцентные фрагменты:

- `.statement__highlight--fill` («правильный») — заливка `#E31E24`, белый текст, `border-radius: 2px`, padding `4px 14px` (идентично `.growth__highlight--fill`).
- `.statement__highlight--text` («разговор») — без плашки, `color: #E31E24` (идентично `.growth__highlight--text`).
- `.statement__highlight--circle` («больше») — текст остаётся чёрным (`#0A0A0C`), оборачивается в инлайн-`<svg>` с неровным овальным `path` (красный `stroke`, `fill: none`, `stroke-linecap: round`) — имитация обводки маркером от руки. `<svg>` позиционируется `position: absolute`, перекрывая слово с небольшим запасом (по аналогии с `.growth__decor` — это разовый декоративный элемент, без отдельного файла-ассета).

## Декор

Слева от текста — вертикальная полоса `.statement__bar` (`#E31E24`, ширина 4px, высота по тексту, `align-self: stretch` во flex-контейнере `.statement__inner`) — по аналогии с `.growth__bar`, но без точечной сетки (на референсе её нет, только полоса).

**Responsive:** полоса видна на всех брейкпоинтах (на референсе она присутствует и на мобильном макете) — `display` не меняется по медиа-запросам, только размер текста.

## Очистка

Удаляются:
- Разметка `<section id="manifesto" class="manifesto">...</section>` (index.html).
- CSS-правила `.manifesto`, `.manifesto p:first-child`, `.manifesto__quote` (css/styles.css).

Проверено: `id`/`class` `manifesto` больше нигде не используются (нет ссылок в навигации, JS или других стилях — только в исторических файлах `docs/superpowers/plans/2026-06-19-pobeda-landing.md` и спеке growth-блока, которые не трогаем).

## Вне скоупа

- Нет новой JS-логики и нет интерактивности — блок полностью статичен.
- Не меняется навигация в `<header>` (новый блок не добавляется в якорное меню — как и старый манифест).
- Декоративные SVG/полоса — только инлайн-код, без новых файлов-ассетов.
