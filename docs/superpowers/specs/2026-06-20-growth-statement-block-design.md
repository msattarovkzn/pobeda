# Дизайн: блок-врезка «Бизнес растёт быстрее...»

## Что добавляем

Новая статичная секция-врезка между блоком «Основатель» (`#founder`) и блоком «Манифест» (`#manifesto`) в `index.html`. Источник — присланный пользователем референс-баннер (белый фон, акцентные плашки), адаптированный под тёмную палитру сайта.

## Контент

Текст используется как на референсе, без изменений:

> Бизнес **растёт** [плашка-заливка]
> **быстрее** [акцентный текст] там, где рядом
> люди **сильнее тебя** [тёмная плашка].

Без CTA-кнопки — секция работает как эмоциональная связка между «Основателем» и «Манифестом», аналогично существующему блоку `.manifesto`, у которого тоже нет кнопки.

## Разметка

```html
<section id="growth" class="growth">
  <div class="growth__bg" aria-hidden="true"></div>
  <div class="container growth__inner">
    <div class="growth__decor" aria-hidden="true">
      <span class="growth__dots"></span>
      <span class="growth__bar"></span>
    </div>
    <p class="growth__text">
      Бизнес <span class="growth__highlight growth__highlight--fill">растёт</span><br>
      <span class="growth__highlight growth__highlight--text">быстрее</span> там, где рядом<br>
      люди <span class="growth__highlight growth__highlight--dark">сильнее тебя</span>.
    </p>
  </div>
</section>
```

## Визуальный стиль

Секция повторяет уже существующий в проекте паттерн «локально красно-чёрного» блока (как `.hero` и `.founder`): переопределяет `--color-bg`, `--color-bg-alt`, `--color-gold: #E31E24`, `--color-text`, `--color-text-muted`, `--font-serif` внутри своего селектора, плюс `.growth__bg` с радиальным красным свечением (по аналогии с `.hero__bg` / `.founder__bg`) для визуальной связки с этими двумя блоками.

Шрифт — Inter, `font-weight: 800`, без капса (как в референсе), размер между `.hero__subtitle` и `.hero__title` (≈1.75rem на мобильных → 2.25rem от 768px).

Три акцентных фрагмента:

- `.growth__highlight--fill` («растёт») — заливка `var(--color-gold)` (красный), белый текст, `border-radius: 2px` (как у `.btn`), небольшой padding.
- `.growth__highlight--text` («быстрее») — без плашки, просто `color: var(--color-gold)`, тот же вес шрифта.
- `.growth__highlight--dark` («сильнее тебя») — плашка на `var(--color-bg-alt)` (#161618 — тот же токен, что у `.card` / `.modal__dialog`) с тонкой полупрозрачной красной рамкой (`rgba(227, 30, 36, 0.25)`), белый текст. Чистый чёрный (`#000`) не используется — на странице с фоном `#0A0A0C` он был бы почти неотличим от фона.

## Декор

Левая декоративная колонка — точечная сетка 3×3 (чистый CSS `radial-gradient`, без отдельных файлов/SVG) + вертикальная полоса (`var(--color-gold)`, тонкая, на высоту текстового блока), по аналогии с `.founder__name-line`.

**Responsive:** декоративная колонка скрыта на мобильных (`<768px`) — `.growth__decor { display: none; }`, текст идёт без неё, по левому краю. От 768px колонка и текст выстраиваются в строку через flex (аналогично `.founder__inner` на десктопе).

## Вне скоупа

- Нет новой JS-логики и нет интерактивности — блок полностью статичен, автоматизированных тестов не требует (существующие `tests/apply.test.js` и `tests/main.test.js` покрывают форму заявки и API, к этому блоку не относятся).
- Не меняется навигация в `<header>` (новый блок не добавляется в якорное меню, так как пользователь не упомянул это и блок не является самостоятельным «разделом» с собственной темой — он скорее акцентная пауза в нарративе).
- Декоративные точки/полоса — только CSS, без новых файлов-ассетов.
