# Дизайн: блок-врезка «Бизнес растёт быстрее...»

## Что добавляем

Новая статичная секция-врезка между блоком «Основатель» (`#founder`) и блоком «Манифест» (`#manifesto`) в `index.html`. Источник — присланный пользователем референс-баннер (белый фон, акцентные плашки).

**Решение по итогам визуальной проверки мокапов:** блок воспроизводит цветовую схему референса буквально (белый фон, чёрный текст), без адаптации под тёмную палитру остального сайта — осознанный разрыв с общей тёмной темой, подтверждённый пользователем.

## Контент

Текст используется как на референсе, без изменений:

> Бизнес **растёт** [плашка-заливка]
> **быстрее** [акцентный текст] там, где рядом
> люди **сильнее тебя** [тёмная плашка].

Без CTA-кнопки — секция работает как эмоциональная связка между «Основателем» и «Манифестом», аналогично существующему блоку `.manifesto`, у которого тоже нет кнопки.

## Разметка

```html
<section id="growth" class="growth">
  <div class="container growth__inner">
    <div class="growth__decor" aria-hidden="true">
      <span class="growth__dots"></span>
      <span class="growth__bar"></span>
    </div>
    <p class="growth__text">
      Бизнес <span class="growth__highlight growth__highlight--fill">растёт</span><br>
      <span class="growth__highlight growth__highlight--text">быстрее</span> там, где рядом<br>
      люди <span class="growth__highlight growth__highlight--dark">сильнее тебя</span>.
    </p>
  </div>
</section>
```

Секция — обычный полноширинный блок (как `.hero`, `.founder`, `.manifesto`): фон задаётся на `.growth`, контент ограничен через `.container` — без дополнительных full-bleed хаков, это и так стандартное поведение секций на странице.

## Визуальный стиль

- Фон секции: `#F7F7F8` (тёплый белый, как на референсе).
- Текст: `#0A0A0C` (почти чёрный), `font-family: var(--font-sans)` (Inter), `font-weight: 800`, без капса.
- Размер текста — крупный, по аналогии с `.hero__title`: `2rem` на мобильных → `2.75rem` от 768px → `3.5rem` от 1024px (совпадает с десктопным размером `.hero__title`, чтобы блок читался как сопоставимое по весу высказывание).
- Вертикальный паддинг — `var(--space-6)`, как у соседних секций.

Три акцентных фрагмента (цвета — литералы, без переопределения `--color-gold`, так как это разовая чёрно-белая секция, не часть «красно-чёрного» паттерна hero/founder):

- `.growth__highlight--fill` («растёт») — заливка `#E31E24`, белый текст, `border-radius: 2px`, padding `4px 14px`.
- `.growth__highlight--text` («быстрее») — без плашки, `color: #E31E24`, тот же вес шрифта.
- `.growth__highlight--dark` («сильнее тебя») — плашка `#16161A` (тёмная, почти чёрная — не чистый `#000`, чтобы не выглядело пустой дыркой), белый текст, тот же padding/radius.

## Декор

Левая декоративная колонка — точечная сетка 3×3 (чистый CSS `radial-gradient`, без отдельных файлов/SVG, цвет `#E31E24`) + вертикальная полоса (`#E31E24`, ширина 4px, высота по тексту).

**Responsive:** декоративная колонка скрыта на мобильных (`<768px`) — `.growth__decor { display: none; }`, текст идёт без неё, по левому краю. От 768px колонка и текст выстраиваются в строку через flex (decor слева, текст справа, `align-items: center`).

## Вне скоупа

- Нет новой JS-логики и нет интерактивности — блок полностью статичен, автоматизированных тестов не требует (существующие `tests/apply.test.js` и `tests/main.test.js` покрывают форму заявки и API, к этому блоку не относятся).
- Не меняется навигация в `<header>` (новый блок не добавляется в якорное меню — это акцентная пауза в нарративе, а не самостоятельный раздел).
- Декоративные точки/полоса — только CSS, без новых файлов-ассетов.
