# Бизнес-клуб «ПОБЕДА» — лендинг. Дизайн-спека

> Файл-спека для проекта, утверждённая перед началом реализации. Источник требований — `инструкция.md`. Этот файл — рабочее ТЗ: архитектура, тексты, визуальная концепция, техническая реализация.

## 1. Суть проекта

- **Что:** лендинг закрытого платного бизнес-клуба «ПОБЕДА» для предпринимателей-мужчин в Уфе.
- **Формат клуба:** закрытый, платный, вступление только после личного собеседования. Цена не публикуется — обсуждается на собеседовании.
- **Единственная цель сайта:** заявка на собеседование.
- **Источник трафика:** преимущественно Instagram (реклама/био) → **mobile-first** приоритет вёрстки, desktop — адаптация.
- **Эмоция, которую должен вызывать сайт:** «Я нашёл своих людей. Здесь меня понимают. Здесь можно расти быстрее и не идти в одиночку.»

## 2. Аудитория и боли

Предприниматели Уфы, без ограничений по обороту и нише.

**Боли:** одиночество предпринимателя, отсутствие сильного окружения, нехватка людей, понимающих бизнес, отсутствие честной обратной связи, сложность принятия решений, нехватка полезных связей, невозможность открыто обсуждать проблемы, дефицит новых идей, поиск партнёров, нехватка поддержки.

**Желания:** окружить себя сильными людьми, ускорить рост бизнеса, найти партнёров, получить опыт других предпринимателей, совместно реализовывать проекты, общаться без показухи, иметь место, где можно быть собой, чувствовать поддержку, развиваться как предприниматель и как человек.

## 3. Тон и брендинг

**Ощущение:** премиальный, мужской, спокойный, сильный, современный, дорогой, без инфоцыганства, без пафоса, без кричащих продаж.

**Тон голоса:** спокойный, уверенный, интеллектуальный, взрослый.

**Запрещённые формулировки:** «успешный успех», «стань миллионером», «элитное сообщество», «лучшие из лучших» — и любые их смысловые аналоги.

**Смысл названия «ПОБЕДА»:** не победа над конкурентами. Победа над одиночеством, над хаосом, над страхами — через окружение и совместное движение вперёд. Ключевая фраза: *«Большие победы редко достигаются в одиночку»*.

**Текущий статус клуба:** запуск с нуля, без участников и отзывов на момент создания сайта. **На сайте не должно быть фейковых отзывов, логотипов участников или цифр-социального доказательства** — это запрещено по этике и противоречит тону «без инфоцыганства». Социальное доказательство заменяется силой идеи, формата и блока основателя.

## 4. Визуальная концепция

| Параметр | Значение |
|---|---|
| Фон | угольно-чёрный, `#0A0A0C` → `#161618` |
| Акцент | насыщенный красный, `#E31E24` |
| Текст | белый (`#FFFFFF` / `#B9B9BD` для вторичного) на тёмном фоне |
| Заголовки | плотный гротеск, жирное/чёрное начертание (Inter 800–900), в hero — капс |
| Текст тела | нейтральный sans (Inter) |
| Иконки/графика | абстрактная минималистичная line-графика в красном — без stock-фото людей и событий (их пока нет) |
| Фото | плейсхолдеры (силуэт/аватар для основателя) и одно реальное фото-портрет в блоке «ПОБЕДА — это:» (см. §5.3) как визуальная опора, без постановочных stock-фото бизнеса в остальных блоках |

Дорогая, мужская, энергичная эстетика без инфоцыганских градиентов «успеха» и эмодзи-иконок. Логотип — отдельный графический файл (плейсхолдер до получения исходника от клиента).

## 5. Архитектура страницы (one-pager + модалка формы)

Порядок блоков (нарратив «боль → решение»):

1. **Header** — лого «ПОБЕДА» слева, якорная навигация (desktop) (стартовый список: «Что входит», «Основатель», «FAQ» — расширяется по мере добавления новых разделов в ходе текущей переработки структуры), кнопка «Записаться» (открывает модалку) и иконки соцсетей (Instagram, Telegram) справа. Мобильная версия — только лого + CTA-кнопка, без гамбургер-меню и без иконок соцсетей.
2. **Hero** (левое выравнивание текста, фоновая графика — геометричные тёмные грани + красный световой акцент)
   - H1 (капс, акцентные слова красным): «ПОБЕДА — клуб для предпринимателей Уфы»
   - Подзаголовок: «Попади в сильное сообщество предпринимателей с доходом от 300 тыс. руб. до 10 млн руб. в мес.» (цифры — фильтр по доходу аудитории, не социальное доказательство участников)
   - CTA: «Записаться на пробную встречу»
   - Микротекст под CTA: «Вступление только после личного разговора»
   - Внизу по центру — анимированная стрелка-шеврон вниз (лёгкая пульсация/смещение), клик скроллит к следующей секции («Узнаёте себя?») — пока не реализовано, вне скоупа текущей правки
3. **«ПОБЕДА — это:»** (вводный блок сразу после Hero, фон `--color-bg-alt` для ритма секций)
   - Заголовок: «ПОБЕДА — это:»
   - Композиция (мобайл: фото → левый столбец → правый столбец; десктоп ≥1024px: левый столбец | фото | правый столбец): по центру — фото-портрет предпринимателя в рамке с золотой обводкой, по краям — два столбца по 3 пункта (золотая галочка + заголовок + строка пояснения)
   - Левый столбец: «Сильное окружение» — рядом предприниматели, которые понимают твой бизнес · «Честная обратная связь» — без вежливой воды, с реальным разбором ситуации · «Новые связи и партнёры» — знакомства, которые превращаются в сделки и решения
   - Правый столбец: «Разборы и мастермайнды» — сложные вопросы обсуждаем до конкретных действий · «Клуб не только про бизнес» — совместный отдых, где легче говорить и о делах, и о жизни · «Рост быстрее, чем в одиночку» — чужой опыт помогает не терять месяцы и деньги
   - Это единственный блок с реальным (не плейсхолдер) фото человека — осознанное исключение из правила «без stock-фото» в §4. Фото иллюстрирует атмосферу клуба и не привязано к имени/цитате — в отличие от блока «Основатель» (п.7), который остаётся текстовым плейсхолдером до получения реальных данных
4. **Боли — «Узнаёте себя?»**
   - Вступление: «Большинство предпринимателей сталкиваются с одним и тем же — даже если внешне всё выглядит благополучно.»
   - 6 карточек (иконка + короткая строка): одиночество в принятии решений · нет окружения, которое понимает бизнес · не с кем обсудить проблему открыто · не хватает честной обратной связи · сложно искать партнёров и нужные связи · не хватает новых идей и энергии для роста
5. **Смысл «ПОБЕДА»** (манифест-блок, крупная типографика на тёмном фоне)
   - «Победа — это не победа над конкурентами. Это победа над одиночеством. Над хаосом. Над страхами. Победа через окружение. Через совместное движение вперёд.»
   - Pull-quote: «Большие победы редко достигаются в одиночку.»
6. **Что входит в клуб** — сетка из 11 пунктов, каждый: золотая line-иконка + заголовок + одна строка пояснения:
   1. Ежемесячные встречи — реальные разговоры, а не созвоны для галочки
   2. Мастермайнды — разбор сложных решений с теми, кто прошёл похожий путь
   3. Закрытый чат — всегда на связи с людьми, которые понимают бизнес
   4. Спортивные мероприятия — энергия и форма наравне с делом
   5. Совместные поездки — больше контекста, больше доверия
   6. Приглашённые эксперты — практический опыт, не теория
   7. Поиск партнёров — нужные связи внутри закрытого круга
   8. Нетворкинг — знакомства, которые работают
   9. Совместный отдых с семьями — клуб не только про бизнес
   10. Помощь в кризисных ситуациях — поддержка, когда она нужнее всего
   11. Разборы бизнеса участников — взгляд со стороны от тех, кому можно доверять
7. **Основатель** (реальные данные; красно-чёрная палитра локально для блока — см. §4)
   - Композиция: мобайл — фото сверху, затем текст; десктоп ≥768px — текст слева, фото справа
   - Декоративная красная кавычка над текстом
   - Текст: «Имею реальный опыт построения бизнеса: доставка еды «Аппетит» с оборотом более 1 млрд рублей в год и строительные проекты в Башкортостане и других регионах России.» — разделитель — «За годы в предпринимательстве понял главное: сильное окружение и наставники позволяют проходить путь быстрее, избегать многих ошибок и достигать большего.» (ключевые фразы — акцентным красным)
   - Имя: «Рустам Гумеров»
   - Подпись: «Предприниматель, основатель бизнес-клуба «Победа»»
   - CTA: «Вступить в клуб»
   - Фото: `assets/founder-photo.webp` (портрет, аналогично фото в блоке «ПОБЕДА — это:» из §5.3)
8. **Формат клуба** — три факта карточками:
   - Закрытый клуб — вступление только после собеседования
   - Платное участие — условия обсуждаются индивидуально на собеседовании
   - Без ограничений по нише и обороту — важна готовность быть частью круга
9. **FAQ**
   - «Сколько стоит участие?» → «Стоимость и условия участия обсуждаются индивидуально на собеседовании.»
   - «Как происходит вступление?» → «Вы оставляете заявку, мы созваниваемся, знакомимся и решаем, подходит ли клуб друг другу.»
   - «Для кого этот клуб?» → «Для предпринимателей Уфы — без ограничений по нише и обороту. Важнее готовность быть открытым и вкладываться в окружение.»
   - «Это про обучение или нетворкинг?» → «Ни то, ни другое в чистом виде. Это среда: встречи, мастермайнды, честная обратная связь и поддержка от людей, которые сами в бизнесе.»
   - «Сколько человек в клубе?» → «Клуб закрытый и небольшой по составу — это принципиально для качества общения.» *(плейсхолдер-формулировка, уточнить после запуска)*

    - Заголовок: «Готовы перестать быть один на один с бизнесом?»
    - Кнопка: «Записаться на пробную встречу»
11. **Footer** — лого «ПОБЕДА», строка «Закрытый клуб предпринимателей. Уфа.», ссылка на Instagram (плейсхолдер `#`), ссылка на политику обработки персональных данных (`privacy.html`), копирайт.

### Модальная форма заявки

- Заголовок: «Запись на пробную встречу»
- Поля: Имя, Телефон или Telegram
- Чекбокс (обязательный): «Даю согласие на обработку персональных данных в соответствии с Политикой обработки персональных данных» — ссылка на `privacy.html`
- Скрытое honeypot-поле для базовой защиты от спам-ботов
- Кнопка: «Отправить заявку»
- Состояние успеха: «Заявка получена. Мы свяжемся с вами в ближайшее время.»
- Состояние ошибки: «Не удалось отправить заявку. Попробуйте ещё раз или напишите нам в Telegram.»
- На мобильных — дополнительно sticky-кнопка «Записаться на пробную встречу» снизу экрана (появляется после скролла мимо hero, скрывается при открытой модалке) — для повышения конверсии трафика из Instagram.

## 6. Техническая реализация

**Стек:** статический сайт, без билд-инструментов и фреймворков.

```
/index.html              — лендинг (все блоки)
/privacy.html            — политика обработки персональных данных (152-ФЗ)
/css/styles.css           — стили, переменные цвета/типографики, mobile-first媒体-запросы
/js/main.js               — модалка, валидация, fetch к /api/apply, sticky CTA, плавный скролл
/api/apply.js             — Vercel serverless-функция (Node.js)
/assets/icons/*.svg       — золотые line-иконки
/assets/about-photo.webp  — фото-портрет для блока «ПОБЕДА — это:»
/assets/founder-photo.webp — фото-портрет основателя для блока «Основатель»
/assets/og-image.jpg      — изображение для Open Graph
/assets/favicon.*         — favicon
/robots.txt, /sitemap.xml
```

**Обработка формы:**
1. Клиент валидирует поля (имя и контакт не пустые, чекбокс отмечен) перед отправкой.
2. `main.js` отправляет `POST /api/apply` с `{ name, contact }`.
3. `api/apply.js` валидирует данные на сервере, формирует сообщение и отправляет его в Telegram через Bot API (`https://api.telegram.org/bot<TOKEN>/sendMessage`) в чат владельца клуба.
4. Токен и chat_id хранятся в переменных окружения Vercel (`TELEGRAM_BOT_TOKEN`, `TELEGRAM_CHAT_ID`), не в коде.
5. Функция возвращает `{ ok: true }` или ошибку; модалка показывает соответствующее состояние.

**Деплой:** Vercel (бесплатный тариф), временный домен `*.vercel.app`. Позже — подключение собственного домена (когда появится).

**SEO:**
- Семантический HTML5 (`header`, `main`, `section`, `footer`), один `H1` в hero, `H2` на каждый блок.
- `<title>`: «Бизнес-клуб «ПОБЕДА» — закрытый клуб предпринимателей Уфы»
- `meta description` под запросы вида «бизнес-клуб Уфа», «клуб предпринимателей Уфа».
- Open Graph / Twitter Card теги (`og:title`, `og:description`, `og:image`, `og:locale=ru_RU`).
- `alt`-тексты у всех иконок/изображений.
- `robots.txt` + минимальный `sitemap.xml` (2 страницы).

**Аналитика:** Яндекс.Метрика — счётчик в `<head>` (ID — плейсхолдер), цель `form_submit`, вызывается при успешной отправке формы.

**Адаптивность:** mobile-first (базовые стили под ~375–428px), `min-width` медиа-запросы для планшета (768px) и десктопа (1024px+).

## 7. Юридический блок (152-ФЗ)

`privacy.html` — типовой шаблон политики обработки персональных данных: оператор (плейсхолдер реквизитов), цели обработки (обработка заявки на собеседование), состав данных (имя, телефон/Telegram), отсутствие передачи третьим лицам кроме технических сервисов (хостинг, Telegram), права субъекта на отзыв согласия, контакт для обращений.

> Это типовой текст для старта, не юридическая консультация. Рекомендуется проверить у юриста перед реальным запуском рекламы.

## 8. Плейсхолдеры — заполнить после запуска

- Графический логотип «ПОБЕДА» (SVG/PNG с прозрачным фоном) — сейчас в header/footer текстовая заглушка.
- Файл `assets/founder-photo.webp` (текст и имя основателя уже актуальны, фото-файл ещё нужно положить в проект).
- Ссылка/ник на Instagram клуба и Telegram-канал клуба (сейчас — `#`).
- `TELEGRAM_BOT_TOKEN` и `TELEGRAM_CHAT_ID` (переменные окружения Vercel).
- ID счётчика Яндекс.Метрики.
- Собственный домен (сейчас — `*.vercel.app`).
- Реквизиты оператора персональных данных в `privacy.html`.
- Точная формулировка про размер клуба в FAQ.

## 9. Вне скоупа (явно не делаем)

- Многостраничная навигация / блог.
- Реальные отзывы/кейсы (их физически нет на старте).
- Личный кабинет участника, оплата онлайн.
- CRM-интеграция (на старте — только Telegram).
